Template
========

$project is a next-generation distribution of Apache Hadoop.

Build the documentation
------------------------

   # Install sphinx

   pip install sphinx sphinx-autobuild recommonmark sphinx_rtd_theme jupyter_sphinx_theme

   cd docs

   make html
